Angular Music Player:

This was a fun, music player assignment given to me by my Netcentric class.
I used Angular CLI, with the help of free-to-use .mp3 files to construct our own, personalized
music streaming application. I also implemented a search bar, where if the user searched up a
letter (upper-case or lower,) then the song that matches the most would show up on the "Top
Songs" page. It can also pause, skip to the next track, or skip to a precious track.

Scavenge:

Scavenge is a virtual reality space scavenging game that uses the Unity engine,
along with C# scripts to run and do tasks. It was constructed as a senior project
for Penn State Harrisburg's Capstone program. As you travel from numerous derelict ships,
your task is to collect or "scavenge" items in hopes to craft and upgrade items
for the player and their ship alike.
